CSS

CSS(Cascading Stlye Sheets)는 HTML로 작성된 문서를 쓰기위한 스타일 시트 언어입니다.

HTML은 문서의 골격을 이룬다고 하면 CSS는 그 골격에 맞는 디자인 작업 언어라고 생각하면 됩니다.

CSS 선택자

CSS선택자를 통해 요소를 선택하고 속성을 줄 수 있습니다.

기본 선택자

종류 예시 이름
태그 선택자 p {color: #fff} p태그를 선택해서 글씨 색을 하얀색으로 설정합니다.
클래스 선택자 .class {color: #fff} class 클래스를 선택하여 글씨 색을 하얀색으로 설정합니다.
아이디 선택자 #id {color: #fff} id 아이디를 선택하여 글씨 색을 하얀색으로 설정합니다.
그룹 선택자 p, div, #id {color: #fff} p, div, #id 를 선택하여 글씨 색을 하얀색으로 설정합니다.
전체 선택자 *{color: #fff} 전체 태그를 선택하여 글씨 색을 하얀색으로 설정합니다.
  • id 선택자
    1. 하나의 태그만 식별하기 위한 선택자입니다.
    2. 하나의 태그에서 하나만 사용할 수 있습니다.
    3. "#"으로 표시합니다.
  • class 선택자
    1. 여러개의 태그를 식별하기 위한 선택자입니다.
    2. 하나의 태그에서 여러개를 사용할 수 있습니다.
    3. "."으로 표시합니다.

링크 가상 클래스

종류 예시 이름
:link a:link {color: #fff} 방문하지 않은 링크의 글씨 색을 하얀색으로 설정합니다.
:visit a:visit {color: #fff} 방문한 링크의 글씨 색을 하얀색으로 설정합니다.
:hover a:hover {color: #fff} 링크 위에 마우스를 올려 놓았을 경우 글씨 색을 하얀색으로 설정합니다.
:active a:active {color: #fff} 링크가 활성화되었을 경우 글씨 색을 하얀색으로 설정합니다.

폼 관련 선택자

종류 예시 이름
:checked input:checked {color: #fff} input 태그가 체크되었을 때 글씨 색을 하얀색으로 설정합니다.
:focus input:focus {color: #fff} 태그에 마우스의 초점이 맞추어졌을 때 글씨 색을 하얀색으로 설정합니다.
:enabled input:enabled {color: #fff} input태그가 사용 가능할때 글씨 색을 하얀색으로 설정합니다.
:disabled input:disabled {color: #fff} input태그가 사용 가능하지 않을때 글씨 색을 하얀색으로 설정합니다.

속성 선택자

종류 예시 이름
[name] a[herf] a태그의 속성 중에 href가 있으면 선택합니다.
[name="value"] a[herf="#header"] a태그 href속성의 값이 '#header'이면 선택합니다.
[name~="value"] a[herf~="web"] a태그 href속성의 값이 'web'을 포함한 요소를 선택합니다.
[name|="value"] a[herf|="web"] a태그 href속성의 값이 'web'이거나 'web-'으로 시작하는 요소를 선택합니다.
[name^="value"] a[herf^="web"] a태그 href속성의 값이 'web'으로 시작하는 요소를 선택합니다.
[name*="value"] a[herf*="web"] a태그 href속성의 값이 'web'으로 포함되어 있는 태그를 선택합니다.
[name$="value"] a[herf$="com"] a태그 href속성의 값이 'com'으로 끝나는 태그를 선택합니다.

CSS 색상

CSS를 통해 색상을 표현하는 방법은 4가지가 있습니다.

16진수 표기법

웹 문서에서 색상을 표현하는 방법은 16진수로 표현하는 방법입니다. 두자리씩 묶어서 #RRGGBB형식으로 표현하며, RR은 빨간색, GG는 초록색, BB는 파란색의 양을 표현합니다. 각 색상의 색이 하나도 섞이지 않은것은 00부터, 가득 섞인것은 FF까지 사용할 수 있습니다.

RGB와 RGBA

웹 문서에서 색상을 표현하는 RGB(255,255,255)는 십진수를 이용하여 표현합니다. 색이 하나도 섞이지 않았을 때에는 0으로 표시하고, 가득 섞였을때에는 255로 표현합니다. 숫자는 색상의 양을 나타내고 a는 투명도(alpha)를 나타냅니다. 투명도 0과 1사이로 표현합니다.

HSL와 HSLA

웹 문서에서 색상을 표현하는 HSL은 색상(hue), 채도(saturation), 밝기(lightness)를 나타냅니다. 색상은 둥글게 배치한 색상환으로 표시하고 0도와 360도에는 빨간색, 120도에는 초록색, 240도에는 파란색이 배치됩니다. 채도가 0%이면 회색톤, 100%이면 순색으로 표시됩니다. 밝기는 0%에서는 가장 어둡고 100%에서는 가장 밝습니다.

색상 이름 표기법

색상 이름을 이용하여 색을 표현할 수 있습니다. 가장 기본적인 16가지를 포함하여 216가지의 색상 이름 표기법이 있습니다.

aqua, black, blue, fnchsia, gray, green, lime, maroon, navy, olive, purple, red, silver, white, yellow, teal (기본적인 16가지 색)

비트맵과 벡터

컴퓨터 그래픽에서 이미지 표현방식은 크게 비트맵과 벡터로 구분됩니다.

  • 비트맵
    • 비트맵(Bitmap)은 비트의 지도(map of bits)란 뜻입니다.
    • 각 픽셀은 R, G, B가 하나의 픽셀을 이루고 그 픽셀이 모여서 하나의 이미지를 만들게 됩니다.
    • 최하위의 레벨에서 고정된 형식의 이미지라서 많은 효율성을 가지게 되고, 사진을 합성하는데 적합합니다.
    • 확대나 축소시에 이미지 손상이 일어나고 고해상도일수록 이미지의 용량이 커집니다.

      • 비트맵을 확대했을때의 이미지

  • 벡터
    • 벡터는 점과 점을 연결해서 이미지를 표현합니다.
    • 두개의 점이 이어져서 선이되고, 선이3개 이상이면 면이됩니다. 이러한 것들이 모여 벡터 이미지를 만들어냅니다.
    • 확대나 축소시에 이미지가 깔끔합니다.

      • 벡터를 확대했을때의 이미지

      포토샵에서 벡터로 표현되어있을때

CSS 선언

HTML 문서에 스타일 선언하는 방법은 3가지 방법이 있습니다.

  • 내부 스타일(Internal Style Sheet) : head태그 안에 선언하는 방법
  • 외부 스타일(External Style Sheet) : 외부 파일로 연결하는 방법
  • 인라인 스타일(Inline Style Sheet) : 태그에 직접 연결하는 방법
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
<!--    내부스타일-->
    <style>
       h1 {color: black;}
    </style>
    
<!--    외부스타일-->
   <link rel="stylesheet" href="경로">
    
</head>
<body>
<!--    인라인스타일-->
    <h1 style="color: black"></h1>
</body>
</html>

상대주소와 절대주소

이미지 또는 파일의 경로를 설정하는 방법은 상대주소와 절대주소가 있습니다.

  • 상대주소 : 현재 위치를 기준으로 한 웹 페이지나 파일의 경로(위치)를 말합니다.
    • 상대주소의 경로의 기준은 현재 웹페이지의 소속 폴더가 기준점입니다.
    • . : 현재 웹페이지가 소속된 폴더(같은 폴더의 다른페이지)
    • .. : 현재 웹페이지의 부모 폴더(더 큰 프로젝트 폴더의 다른페이지)
  • 절대주소 : 특정한 웹 페이지나 파일이 가지고 있는 고유한 경로(위치)를 말합니다.
    • 누구나 알고있는 동일한 위치를 기준으로 절대로 변하지 않는 주소입니다.
    • C:\Atom\a.html : c:디스크안에 있는 Atom폴더의 a.html 파일이 되므로 다른 컴퓨터로 하게되면 나오지 않기 때문에 경로를 바꿔줘야 하는 일이 생깁니다.

이미지 표현 방법

웹 문서에서 이미지를 표현하는 방법은 img태그와 background를 통해 설정할 수 잇습니다.

  • img 태그로 표현하는 방법 : 이미지가 의미가 있는 경우 (로고, 아이콘)
    Alt태그(대체재)가 있어서 좋지만 너무복잡함
  • background 로 표현하는 방법 : 이미지가 의미가 없는 경우 (장식), 시각리더기로 읽으면 인식이 안됨

이미지 스프라이트

아이콘 또는 장식을 위한 이미지 요소들은 스프라이트 기법을 활용하여 파일의 사이즈를 최소화하고 효율성을 높일 수 있습니다.

  • 여러번의 서버 요청을 한번으로 줄일 수가 있습니다.
  • 이미지 수정이나 관리가 간편합니다.
  • 웹 접근성을 준수하기 위해서 IR효과를 설정해야 합니다.

IR 효과

IR 기법은 이미지 대체텍스트를 제공하기 위한 CSS기법으로, 다양한 CSS기법을 사용하여 이미지 대체 텍스트를 제공할 수 있습니다.

Phark Method

의미 있는 이미지의 대체 텍스트를 제공하는 경우: 이미지로 대체할 엘리먼트에 배경이미지를 설정하고 글자는 text-indent를 이용하여 화면 바깥으로 빼내어 보이지 않게 하는 방법

항목 장점 단점
스크린 리더가 읽어줌 O
추가적인 태그 사용 안함 O
CSS on/ Image off시 텍스트 안보임 O

PWA IR

의미 있는 이미지의 대체 텍스트로, 이미지를 off시에도 대체 텍스트를 보여주고자 할때: 이미지로 대체할 엘리먼트에 배경이미지를 설정하고 글자는 span 태그로 감싼후 z-index: -1;을 이용하여 화면에 안보이게 처리하는 방법

Screen Out

대체 텍스트가 아닌 접근성을 위한 숨김 텍스트를 제공할 때 사용합니다.

float: left 로 인한 영역 깨짐 방지법

float을 쓰게 되면 영역의 높이값이 0으로 줄어들기 때문에 영역을 유지하는 방법이 필요합니다.

  1. 깨지는 영역에 똑같이 float: left를 사용합니다. (X: 모든 박스영역에 float을 사용하게 되고 정확하게 깨지는 영역을 알 수 없습니다.)
  2. float의 성질을 차단하는 clear: both를 사용합니다. (X: 깨지는 영역을 정확히 알 수 없습니다.)
  3. float을 사용한 부모 박스 영역에 overflow: hidden을 사용합니다. (△)
  4. clearfix를 사용하여 가상의 영역을 만들어줍니다. (O)

미디어 쿼리(Media Query)

미디어 쿼리는 데스크탑, 노트북, 태블릿 및 휴대폰 등 화면의 크기에 따라서 웹 페이지의 각 요소들에게 다른 스타일을 적용할 수 있습니다.

  • 반응형 웹/ 반응형 디자인은 미디어 쿼리를 사용하여 구현할 수 있습니다.(ie8는 지원안함)
  • link내에서 <link href="css파일 경로" rel="stylesheet" type="text/css" media="미디어 쿼리 조건"> 로 사용할 수 있습니다
  • 특정 조건이 true인 경우에만 CSS 속성 블록을 적용하도록 @media 규칙을 사용합니다.
  • 미디어 쿼리 예제 : mediaqueri
  • @media only all and ( 조건 ) { 조건이 달성될 경우 이곳의 css를 적용함 }
  • only : 미디어쿼리를 지원하지 않는 브라우저가 주어진 스타일을 적용하는 것을 방지 하는 키워드로써 생략 가능합니다. 생략했을 때 기본 값은 only 입니다.

조건부 주석

조건부 주석은 어떠한 조건이 충족되지 않으면 주석처리가 되고, 조건이 충족되면 주석 안의 내용이 브라우저에 렌더링 되게끔 설정합니다.

  • IE10 미만의 IE(익스플로러)에서만 해석해서 실행시킵니다.
  • 크롬 등 다른 브라우저에서는 해당 코드를 주석으로 인식하여 무시합니다.
  • <!--[if 조건]> 처리 할 내용 <![endif]-->

SCSS

SCSS는 SASS와 CSS가 합쳐진 것입니다.

SASS(Syntactically Awesome StyleSheets)는 CSS 전처리기(pre-processor) 로서 CSS의 한계와 단점을 보완하여 보다 가독성이 높고 코드의 재사용에 유리한 CSS를 생성하기 위한 CSS의 확장(extension)입니다.

Sass(SCSS)는 웹에서 직접 동작할 수 없음으로, 최종에는 표준 CSS로 동작해야 하며, 전처리기로 작성 후 CSS로 컴파일해야 합니다.

SCSS 에서 CSS로 변환하는(Compile) 사이트! sassmeister

Attribute

float

float 속성은 블록요소의 정렬 상태를 설정합니다.

  • float: left;
  • float: right;
  • float: none; 정렬안함

width

width 속성은 요소의 가로값을 정의합니다.

height

height 속성은 요소의 세로값을 정의합니다.

text-align

text-align 속성은 텍스트 정렬 방식을 설정합니다.

  • text-align: left;
  • text-align: right;
  • text-align: center; 가운데 정렬
  • text-align: justify; 양쪽 정렬(왼쪽과 오른쪽 둘다 정렬시킴)

*해당 태그의 자식요소들 중에서 inline 속성을 가지는 요소들을 해당 태그의 너비 기준으로 정렬*

text-transform

text-transform 속성은 텍스트를 대문자/소문자로 변경합니다.

  • text-transform: none; 적용안함
  • text-transform: capitalize; 각 단어의 첫 글자만 대문자
  • text-transform: uppercase; 모든 글자 대문자
  • text-transform: lowercase; 모든 글자 소문자

text-decoration

text-decoration 속성은 텍스트의 장식을 설정합니다.

  • text-decoration: none; 텍스트에 선을 긋지 않습니다.
  • text-decoration: line; 텍스트를 underline, overline, line-through으로 선을 그을수 있습니다.
  • text-decoration: color; 텍스트에 그어진 선의 색을 설정합니다.
  • text-decoration: style; 텍스트에 그어진 선의 스타일을 설정합니다.
    • solid; 한 줄
    • double; 두 줄
    • dotted; 점선
    • dashed; 파선
    • wavy; 물결

text-indent

text-indent 속성은 텍스트를 들여쓰기를 설정합니다.

기본 값은 0이고 값이 양수이면 들여쓰기, 음수이면 내어쓰기가 됩니다.

clear

clear 속성은 float 요소의 성질을 차단합니다.

margin

margin 속성은 박스요소의 바깥쪽 여백을 설정합니다.

  • margin-top: 10px -- 요소의 위쪽 바깥여백을 설정
  • margin-right: 11px -- 요소의 오른쪽 바깥여백을 설정
  • margin-bottom: 12px -- 요소의 아랫쪽 바깥여백을 설정
  • margin-left: 13px -- 요소의 왼쪽 바깥여백을 설정
  • margin: 10px 11px 12px 13px -- 위쪽10/ 오른쪽11/ 아랫쪽12/ 왼쪽13 (시계방향)으로 바깥여백을 설정
  • margin: 10px 11px 12px -- 위쪽10/ 왼,오른쪽11/아랫쪽12으로 바깥여백을 설정
  • margin: 10px 11px -- 위,아랫쪽10/ 왼,오른쪽11으로 바깥여백을 설정
  • margin: 10px -- 위,아래,왼,오른쪽 모두 10으로 바깥여백을 설정
  • margin: auto -- 자동값, 기본값
  • margin: 0(px생략가능) auto -- 위,아랫쪽0 왼,오른쪽auto로 바깥여백을 설정(블록구조를 가운데 정렬할때 사용)

padding

padding 속성은 안쪽여백을 설정합니다.

  • margin과 padding은 블록요소에는 위,아래,양옆 다되지만, 인라인요소에서는 양옆에만 적용됩니다.
  • 인라인 요소에서 위,아래의 값을 주고 싶다면 인라인요소를 블록화 해야합니다.

position

position 속성은 요소의 위치를 설정합니다.

  • position: static(기본)
  • position: absolute; 요소의 위치를 절대적으로 설정합니다.

  2. z-index를 사용했을 때

border속성은 테두리 속성을 설정합니다.

  • border-left/top/right/bottom; 테두리 부분지정
  • border-left/top/right/bottom-width; 테두리-부분-의 width값 지정
  • border-left/top/right/bottom-style; 테두리-부분-의 style값 지정(선 모양은 꼭 지정해야 합니다.)
  • border-left/top/right/bottom-color; 테두리-부분-의 선 color값 지정
  • border-image;
  • border-image-outset;
  • border-image-repeat;
  • border-image-slice;
  • border-image-source;
  • border-image-width;
  • border-radius;

overflow

overflow 속성은 내용이 요소의 크기를 벗어났을때의 방식을 설정합니다.

  • overflow: visible; 기본값, 박스를 넘어가도 내용을 보여줍니다.
  • overflow: hidden; 박스를 넘어간 부분은 잘려지고 보이지 않습니다.
  • overflow: scroll; 박스를 넘어가든 넘어가지 않든 스크롤바가 나옵니다.
  • overflow: auto; 박스를 넘어가지 않으면 스크롤바가 나오지 않고, 박스를 넘어갈 때에는 스크롤바가 나옵니다.

list-style

list-style 속성은 ul, ol, li 요소의 리스트 앞에 나오는 블릿 또는 숫자의 스타일을 설정합니다.

  • list-style: none; 머릿글을 표현하지 않습니다.
  • list-style-type: disc, circle, square, lower-roman, upper-roman, lower-alpha, upper-alpha 등이 있습니다.
  • list-style-image: 머릿글로 이미지를 사용할 수 있습니다.

letter-spacing

letter-spacing 속성은 글자 사이의 간격을 설정합니다.

  • 값이 커지면 간격이 커집니다.
  • 값이 음수일 수 있는데, 음수를 값으로 하는 경우 글자가 겹칠 수 있습니다.
  • 마이크로소프트 엣지나 인터넷 익스플로러에서는 단어 사이의 간격을 음수로 정해도 글자가 겹치지 않을 정도로만 간격이 줄어듭니다.

text-overflow

text-overflow 속성은 박스안의 내용물이 넘칠때의 텍스트를 설정합니다.

text-overflow를 구현하기 위한 조건이 있습니다.

  • width 또는 height가 고정적일 것
  • overflow: hidden, scroll, auto 를 사용해 영역을 감출 것
  • 아래줄로 내려가는 것을 막기위해 white-space: nowrap 등이 필요함
  • text-overflow: clip; 기본값입니다. 텍스트를 자릅니다.
  • text-overflow: ellipsis; 잘린 텍스트가 줄임표'...'로 표현됩니다.

white-space

white-space 속성은 어떤 요소(element) 안의 공백(whitespace)이 어떻게 처리될지를 설정합니다.

  • white-space: normal; 연속된 공백과 줄 바꿈은 메꾸어져서 하나의 공백으로 표시됩니다. 소스(Source)내의 줄 바꿈 문자도 공백 문자로써 처리됩니다. 긴 줄을 줄바꿈 합니다.
  • white-space: nowrap; 연속된 공백 문자는 통합하지만, 줄바꿈은 생기지 않습니다.
  • white-space: pre; 연속된 공백을 포함한 원문 그대로 출력하고, 줄바꿈은 생기지 않습니다.
  • white-space: pre-wrap; 연속된 공백은 그대로 유지되고, 긴 줄을 줄바꿈 합니다.
  • white-space: pre-line; 연속된 공백 문자는 통합하지만, 긴 줄을 줄바꿈 합니다.

!important

!important 속성은 최우선 순위를 설정합니다.

CSS는 같은 속성을 여러 번 정의했을 때 나중에 설정한 값이 적용됩니다. 만약 나중에 설정한 값이 적용되지 않게 하기 위해서는 속성값뒤에 !important; 를 붙입니다.

opacity

opacity 속성은 투명도를 설정합니다.

  • 0~1 사이의 값을 넣으며, 숫자가 낮을수록 투명화됩니다.
  • opacity를 사용하게 되면 요소의 배경과 내용 모두를 투명하게 만듭니다.
  • 모두 투명하게 하지 않으려면 배경색에 RGBA를 사용해서 투명도를 조절합니다.

visibility

visibility 속성은 문서의 레이아웃을 변경하지 않고 요소를 보이거나 숨깁니다.

  • visibility: visible; 박스가 보임
  • visibility: hidden; 박스가 보이진 않지만 공간은 유지함 (영역을 계속 차지함)

font-weight

font-weight 속성은 글자의 굵기를 설정합니다.

font-weight로 나타낼 수 있는 굵기는 글꼴에 따라 다릅니다.

  • font-weight: normal; 기본값 숫자 400과 같습니다
  • font-weight: bold; 굵은 굵기, 700과 같습니다.
  • font-weight: bolder; 상속된 값보다 굵은 굵기입니다.
  • font-weight: lighter; 상속된 값보다 얇은 굵기입니다.
  • font-weight: number: 100, 200, 300, 400, 500, 600, 700, 800, 900; normal 과 bold 이외를 제공하는 폰트를 위한 굵기입니다.

placeholder

placeholder 속성은 텍스트 필드 내에 있는 짧은 도움말을 설정합니다.

vertical-align

vertical-align 속성은 인라인 블록 등을 포함한 모든 인라인요소의 수직 정렬을 설정합니다.

  • vertical-align : base; 기본값, 부모 요소의 baseline 에 맞춥니다.
  • vertical-align : sub; 아래첨자에 기준을 맞춰서 정렬합니다.
  • vertical-align : super; 위첨자 기준으로 맞춰서 정렬합니다.
  • vertical-align : text-top; 요소 상단을 부모 font 의 상단에 맞춥니다.
  • vertical-align : text-botom; 요소 하단을 부모 font 의 하단에 맞춥니다.
  • vertical-align : top/middle/bottom; 부모 요소의 상단/중앙/하단 으로 맞춥니다.
  • vertical-align : 길이값/%; 0px/0%는 baseline과 같은 값입니다. 0을 기준으로 조절합니다.

cursor

cursor 속성은 해당 태그 위에 위치하는 마우스 커서의 모양을 설정합니다.

  • auto: 자동
  • default : 기본값 (화살표)
  • pointer : 손가락 모양
  • wait : 로딩
  • crosshair : 십자 모양, 주로 비트맵 선택할 때
  • move : 화살표 십자 모양, 옮길 수 있음을 표시
  • help : 물음표나 풍선 모양, 도움말
  • resize : 사이즈 조절 커서

box-sizing

box-sizing 속성은 박스의 크기를 어떤 것을 기준으로 계산할지를 설정합니다.

width와 height에 padding과 border를 포함할지 여부를 결정하는 속성입니다.

일반적으로 Width값은 width + padding-left + padding-right + border-left + border-right 이라서 실제너비 값은 달라집니다.
그래서 box-sizing: border-box; 이 속성을 넣으면 선언한 width값에 padding을 주더라도 자동으로 실제 컨텐츠 부분의 width가 줄어들면서 실제 보여지는 값은 선언한 width값이 됩니다.

  • content-box : 콘텐트 영역을 기준으로 크기를 정합니다. (기본적)
  • border-box : 테두리를 기준으로 속성값이 padding과 border를 포함하여 크기를 정합니다.

box-orient

box-orient 속성은 flexible박스 안의 자식요소의 배치방향을 설정합니다.

flexible box라는 것은, 사이즈와 배치를 상대적으로 지정할 수 있는 유연한 box를 말합니다.
일반 박스를 flexible box로 바꾸기 위해서는 먼저 display: box 를 지정합니다.

  • vertical : 위에서 아래로 수직으로 정렬
  • horizontal : 왼쪽에서 오른쪽으로 수평으로 정렬
  • inline-axis : inline속성으로 정렬
  • block-axis : block속성으로 정렬

line-clamp

line-clamp 속성은 여러줄에서의 말줄임표를 설정합니다.

기본 말줄임은 text-overflow: ellipsis; 을 사용하였지만 여러줄에서는 -webkit-line-clamp 를 사용하여 말줄임표를 사용합니다.
아직은 웹킷 기반 브라우저는 가능하지만, 웹킷 기반이 아닐 경우에는 불가능합니다.

  • -webkit-line-clamp : 표현하고 싶은 줄 수;

srcset

srcset 속성은 브라우저에 사용할 이미지들과 그 이미지들의 원본 크기를 설정합니다.

화면 너비(screen width)를 기준으로 image를 정의합니다.

<img src="/img/people_575.jpg" alt="people" //기본
srcset="img/people_400.png 400w,<!--뷰포트 너비가 400px 이하일 때-->
img/people_800.png 2x <!--밀도가 2배 이상일 때-->">

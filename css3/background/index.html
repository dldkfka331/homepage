Background

문법

background-image : url | none | linear-gradient | radial-gradient | repeating-liner-gradient | repeating-radial-gradient;

종류 설명
url background-image 주소를 설정합니다.
none background-image 주소를 설정하지 않습니다.
linear-gradient 선형 그라데이션 background를 설정합니다.
radial-gradient 원형 그라데이션 background를 설정합니다.
repeating-liner-gradient 반복 선형 그라데이션 background를 설정합니다.
repeating-radial-gradient 반복 원형 그라데이션 background를 설정합니다.

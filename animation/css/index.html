문법

Animation

animation : name | duration | timing-function | delay | iteration | derection | fill-mode | play-state

종류 설명
name @keyframe에 지정된 이름을 설정합니다.
duration 애니메이션이 실행되는 시간을 설정합니다.
timing-function 애니메이션 키프레임 움직임을 설정합니다.
delay 애니메이션이 시작되기 전 까지의 시간을 설정합니다.
iteration 애니메이션 반복 횟수를 설정합니다.
direction 애니메이션 방향을 설정합니다.
fill-mode 애니메이션이 끝나고 난 뒤 어떤 값을 적용할 지 설정합니다.
play-state 애니메이션 실행 상태를 설정합니다.

Transition

transiton : property | duration | timing-function | delay

종류 설명
property 트렌지션을 적용할 CSS속성 대상을 설정합니다.
duration 트렌지션의 작동 시간을 설정합니다.
timing-function 트렌지션 움직임 효과를 설정합니다.
delay 트렌지션이 시작되기 전 까지의 시간을 설정합니다.

Timing-function

종류 설명
linear 움직임 효과를 일정한 간격으로 설정합니다. (기본값)
ease 처음에는 서서히 가속하고, 마지막 부분에서 급격히 감속합니다.
ease-in 처음에는 천천히 시작하고, 마지막에 가속합니다.
ease-out 처음에는 최대 속도로 시작하고, 마지막에 감속합니다.
ease-in-out 처음에는 제로 속도로 시작하고, 중간지점에서 최고 속도로 움직이고, 마지막에 서서히 감속합니다.
step-start 애니메이션을 시작점에서 설정합니다.
step-end 애니메이션을 끝점에서 설정합니다.
steps(int, start/end) 애니메이션을 단계별로 설정합니다.
cubic-bezier(n,n,n,n) 버지니아 곡선 값을 만들어서 설정합니다.

버지니아 곡선 : cubic-bezier

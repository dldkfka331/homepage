레이어 마스크

클리핑 마스크

클리핑 마스크는 레이어 마스크와 달리 상위 레이어의 이미지를 하위 레이어에 클리핑하여 레이어 모양으로 잘라낸 것처럼 나타내는 기능입니다.

필터 효과

Channels패널열기 -> 새로운 Channel 만들기 -> Marquee툴로 영역잡기 -> 선택영역에 색채우기 -> blur(Gaussian blur)로 효과주기 -> Pixelate(Color Halftone)으로 효과주기 -> 새로운channel에 ctrl+클릭하기 -> 선택영역반전해서 색채우기

레이어 스타일 적용

글씨Layer에 효과주기 -> 다른글씨 만들어서 레이어 스타일 복사하기
